
# COMBAT_eQTL

This repository contains the code and data used for the COMBAT eQTL study, including scripts for generating figures, performing analyses, and processing gene expression and genotype data.

## Repository Structure

- `data/` – Data for plotting main and supplementary figures.  
- `figures/` – Generated plots for main and supplementary figures.  
- `make_manuscript_figures.R` – Script to recreate figures from the manuscript.
